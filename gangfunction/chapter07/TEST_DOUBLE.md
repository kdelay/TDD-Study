# 테스트 더블의 필요성 
## 테스트 더블이란?
- 실제 객체를 대신해서 테스팅에서 사용하는 모든 방법
## 테스트 더블을 선언하는 방법
- 자바에서는 Mockito같은 모킹 라이브러리를 사용합니다.
```java
// 어노테이션을 사용하는 방법
@Mock
private Game game;

```
```java
// 직접 선언하는 방법
private Game game = mock(Game.class);
```
## 테스트 더블을 사용하는 이유
- 의존성을 격리시켜 원하는 관심사에 집중하는 코드의 작성 때문입니다.
## 테스트 더블의 사용순서
1. 테스트 대상 식별
2. 의존성 식별
3. 테스트 더블 생성
4. 스텁 설정
5. 테스트 실행
6. 결과 검증
7. 테스트 정리
